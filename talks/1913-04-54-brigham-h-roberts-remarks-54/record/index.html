Roberts's real address, Elder Orson F. Whitney's address, and the closing 'Authorities Sustained' list, all misattributed to Roberts under one placeholder title. Manually de-interleaved from the DjVu text using the true reading order (confirmed by running heads); Hyrum G. Smith's address is now its own file (1913-04-53), and Whitney's will be its own file. This file now contains only Roberts's true, complete address on the fast-day welfare system.

Collated word-by-word against the original scan.

Confirmed boundaries: begins immediately after Patriarch Hyrum G. Smith's address with '(Of the First Council of Seventy.) Several times during this conference...' and ends '...The Lord bless you. Restored the full talk, including the 1855 drought/grasshopper famine history and the quoted Church fast-day regulations, de-interleaved and reflowed from the OCR. Correction (continued re-ingest pass): date/session were mislabeled 1913-04-05/Second Day Afternoon Session; independently re-tracing the full true reading order confirms this leaf falls in the Third Day (Sunday, April 6) Closing Session. Corrected accordingly; no body text changed.
