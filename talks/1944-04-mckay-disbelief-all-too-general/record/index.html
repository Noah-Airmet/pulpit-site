◕Authoritative print

Transcribed from the OCR of the official printed Conference Report of the One Hundred Fourteenth Annual Conference (April 6, 7, 9, 1944), scanned by the Church History Library and hosted at the Internet Archive (conferencereport1944a). The page images are two-column; reading order was reconstructed from the DjVu word coordinates (XY-cut), then cleaned and corrected against the printed text: running heads, page numbers, and line-break hyphenation removed; drop-cap and obvious OCR errors fixed. Heavyweight review (cursor-agent) collated the discourse against the IA djvu OCR of the same CHL scan; it occupies printed pp. 119-126 (image leaf = printed page + 1, verified against the IA scandata page map, e.g. leaf 8 = printed p. 7). Running heads, page numbers, the printed speaker heading, and sidehead capitalization were reconciled to the printed report. source_archived_url points to the Internet Archive item itself (a Church History Library preservation copy); a dedicated Wayback Machine snapshot still needs to be created with an authenticated Save Page Now.
